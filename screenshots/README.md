# Screenshots

Esta carpeta contiene las capturas de pantalla generadas por la aplicación TPV El Haido.

## Uso

Las capturas se generan automáticamente cuando:
- Estás en modo debug (`config.debug.enabled = true`)
- Haces clic en el botón de la cámara en la esquina inferior derecha
- Las capturas se guardan con nombres como `01_home.png`, `02_settings.png`, etc.

## Estructura de nombres

- `01_login.png` - Pantalla de login
- `01_home.png` - Inicio
- `02_settings_usuarios.png` - Ajustes > Usuarios
- `03_products.png` - Productos
- `04_customers.png` - Clientes
- `05_newOrder.png` - Nueva Comanda
- `06_orderHistory.png` - Historial
- `07_aeatInvoices.png` - Facturas AEAT
- `08_settings_verifactu.png` - Ajustes > VERI*FACTU
- `09_aeatInvoices_detail.png` - Detalle de factura
- `10_settings_about.png` - Ajustes > Acerca de
- `11_settings.png` - Ajustes (general)
- `12_settings_appearance.png` - Ajustes > Apariencia

## Nota

Los archivos PNG/JPG/JPEG se ignoran por git para no committer capturas accidentales.
Solo se debe committer capturas intencionales para documentación.
