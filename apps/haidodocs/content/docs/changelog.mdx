## Formato

El formato sigue [Keep a Changelog](https://keepachangelog.com/es-ES/1.0.0/):

- **Added**: Nuevas funcionalidades
- **Changed**: Cambios en funcionalidades existentes
- **Deprecated**: Funcionalidades que serán eliminadas próximamente
- **Removed**: Funcionalidades eliminadas
- **Fixed**: Corrección de errores
- **Security**: Actualizaciones de seguridad

## [0.1.0] - 2024-XX-XX

### Added
- Sistema de login con PIN por usuario
- Gestión completa de productos y categorías
- Gestión de pedidos/comandas
- Gestión de clientes con datos fiscales
- Historial de pedidos con filtros
- Integración AEAT VERI*FACTU
- Impresión de tickets térmicos ESC/POS
- Sistema de temas claro/oscuro
- Múltiples modos de almacenamiento (SQLite, HTTP, IndexedDB)
- Soporte multiplataforma (Windows, macOS, Linux, Raspberry Pi)
- Sistema de actualizaciones automáticas

### Technical
- Backend Tauri 2.0 con Rust
- Frontend React 19 + TypeScript
- Estado global con Zustand + Immer
- UI con Radix UI + shadcn/ui + Tailwind CSS
- Base de datos SQLite embebida
- Sidecars para AEAT y impresora térmica
