## Instalacion Rapida

<Tabs items={['Windows', 'macOS', 'Linux', 'Raspberry Pi']}>
<Tab value="Windows">

```bash title="Opcion 1: Instalador MSI"
# Descarga y ejecuta TPV.El.Haido_x.x.x_x64-setup.msi
```

```bash title="Opcion 2: Ejecutable portable"
# Descarga y ejecuta TPV.El.Haido_x.x.x_x64.exe
```

<Callout type="warning">
  **Windows SmartScreen**: Haz clic en "Mas informacion" → "Ejecutar de todas formas"
</Callout>

</Tab>
<Tab value="macOS">

```bash title="Intel Mac"
# Descarga TPV.El.Haido_x64.dmg
# Arrastra a Aplicaciones
```

```bash title="Apple Silicon (M1/M2/M3/M4)"
# Descarga TPV.El.Haido_aarch64.dmg
# Arrastra a Aplicaciones
```

<Callout type="info">
  Primera vez: Clic derecho → Abrir para evitar Gatekeeper
</Callout>

</Tab>
<Tab value="Linux">

```bash title="Ubuntu/Debian (.deb)"
wget https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/tpv-el-haido_amd64.deb
sudo dpkg -i tpv-el-haido_amd64.deb
sudo apt-get install -f
```

```bash title="Cualquier distro (AppImage)"
wget https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/tpv-el-haido.AppImage
chmod +x tpv-el-haido.AppImage
./tpv-el-haido.AppImage
```

</Tab>
<Tab value="Raspberry Pi">

```bash title="Raspberry Pi OS 64-bit"
# Verificar arquitectura (debe ser aarch64)
uname -m

# Instalar
wget https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/tpv-el-haido_arm64.deb
sudo dpkg -i tpv-el-haido_arm64.deb
sudo apt-get install -f
```

<Callout type="tip">
  Usa el tema High Contrast para mejor rendimiento en Pi
</Callout>

</Tab>
</Tabs>

## Requisitos del Sistema

| Requisito | Minimo | Recomendado |
|-----------|--------|-------------|
| **RAM** | 512 MB | 1 GB+ |
| **Almacenamiento** | 100 MB | 500 MB |
| **Pantalla** | 800×600 | 1024×768+ |
| **Internet** | Opcional | Para VERI*FACTU |

## Verificar Instalacion

1. Abre TPV El Haido
2. Deberas ver la pantalla de login o el asistente de configuracion
3. Sigue los [Primeros Pasos](/docs/guia-usuario/primeros-pasos)

## Actualizaciones Automaticas

TPV El Haido incluye actualizaciones automaticas:

1. La app comprueba nuevas versiones al iniciar
2. Recibiras una notificacion cuando haya actualizacion
3. Un clic para actualizar desde **Ajustes → Acerca de**

Historial de cambios: [Changelog](/docs/changelog)
