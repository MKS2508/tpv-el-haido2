# Supported Platforms

TPV El Haido supports multiple platforms thanks to Tauri. This guide covers compilation targets and platform-specific considerations.

## Platform Matrix

| Platform | Target | Architecture | Status |
|----------|--------|--------------|--------|
| Windows 10/11 | `x86_64-pc-windows-msvc` | x64 | ✅ Production |
| macOS Intel | `x86_64-apple-darwin` | x64 | ✅ Production |
| macOS Apple Silicon | `aarch64-apple-darwin` | ARM64 | ✅ Production |
| Linux x64 | `x86_64-unknown-linux-gnu` | x64 | ✅ Production |
| Linux ARM64 | `aarch64-unknown-linux-gnu` | ARM64 | ✅ Production |
| Raspberry Pi 4/5 | `aarch64-unknown-linux-gnu` | ARM64 | ✅ Production |

## Requirements by Platform

### Windows

| Requirement | Version |
|-------------|---------|
| Windows | 10 or later |
| WebView2 | Included in Windows 11, installable on Windows 10 |
| Visual Studio | Build Tools 2019+ |

### macOS

| Requirement | Version |
|-------------|---------|
| macOS | 11 Big Sur or later |
| Xcode | Command Line Tools |
| Rust | With aarch64 target for Apple Silicon |

### Linux

| Requirement | Package |
|-------------|---------|
| WebKit2GTK | `libwebkit2gtk-4.1-0` |
| GTK3 | `libgtk-3-0` |
| libappindicator | For system tray |

```bash
# Ubuntu/Debian
sudo apt install libwebkit2gtk-4.1-dev libgtk-3-dev libappindicator3-dev

# Fedora
sudo dnf install webkit2gtk4.1-devel gtk3-devel libappindicator-gtk3-devel
```

## Native Compilation

### Windows

```bash
# From Windows
bun run tauri build

# Output:
# src-tauri/target/release/bundle/
# ├── msi/TPV El Haido_x.x.x_x64_en-US.msi
# └── nsis/TPV El Haido_x.x.x_x64-setup.exe
```

### macOS

```bash
# From macOS
bun run tauri build

# Output:
# src-tauri/target/release/bundle/
# ├── dmg/TPV El Haido_x.x.x_x64.dmg      # Intel
# └── dmg/TPV El Haido_x.x.x_aarch64.dmg  # Apple Silicon
```

### Linux


## Cross-Compilation

### For Raspberry Pi (from Linux x64)

<Steps>

### Install ARM64 toolchain

```bash
# Ubuntu/Debian
sudo apt install gcc-aarch64-linux-gnu

# Configure Rust
rustup target add aarch64-unknown-linux-gnu
```

### Configure linker

Create or edit `~/.cargo/config.toml`:

```toml
[target.aarch64-unknown-linux-gnu]
linker = "aarch64-linux-gnu-gcc"
```

### Compile

```bash
bun run build
bun run tauri build --target aarch64-unknown-linux-gnu
```

### Transfer to Raspberry Pi

```bash
scp src-tauri/target/aarch64-unknown-linux-gnu/release/bundle/deb/*.deb pi@raspberrypi:~
```

</Steps>

### For macOS Universal (Intel + Apple Silicon)

```bash
# From macOS with Apple Silicon
rustup target add x86_64-apple-darwin

# Build for both architectures
bun run tauri build --target universal-apple-darwin
```

## Raspberry Pi

### Requirements

| Requirement | Specification |
|-------------|---------------|
| Model | Raspberry Pi 4 or 5 |
| System | Raspberry Pi OS (64-bit) |
| RAM | 2GB minimum, 4GB recommended |
| Storage | 8GB minimum |

### Installation

```bash
# Download ARM64 .deb
wget https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/tpv-el-haido_arm64.deb

# Install
sudo dpkg -i tpv-el-haido_arm64.deb
sudo apt-get install -f
```

### RPi Optimizations

```bash
# Increase GPU memory (raspi-config)
sudo raspi-config
# → Performance Options → GPU Memory → 256

# Disable compositor if laggy
# Raspberry Pi OS → Preferences → Raspberry Pi Configuration → Display
# → Disable compositor
```

### Autostart on Raspberry Pi

To start TPV El Haido automatically:

```bash
# Create autostart file
mkdir -p ~/.config/autostart
cat > ~/.config/autostart/tpv-el-haido.desktop << EOF
[Desktop Entry]
Type=Application
Name=TPV El Haido
Exec=tpv-el-haido
EOF
```

### Kiosk Mode

To use the RPi as a dedicated terminal:

```bash
# Edit /etc/xdg/lxsession/LXDE-pi/autostart
sudo nano /etc/xdg/lxsession/LXDE-pi/autostart

# Add at the end:
@tpv-el-haido

# Optional: hide cursor after inactivity
@unclutter -idle 0.5 -root
```

## CI/CD with GitHub Actions

### Release Workflow

```yaml
# .github/workflows/release.yml
name: Release

on:
  push:
    tags:
      - 'v*'

jobs:
  build:
    strategy:
      matrix:
        include:
          - os: windows-latest
            target: x86_64-pc-windows-msvc
          - os: macos-latest
            target: x86_64-apple-darwin
          - os: macos-latest
            target: aarch64-apple-darwin
          - os: ubuntu-latest
            target: x86_64-unknown-linux-gnu
          - os: ubuntu-latest
            target: aarch64-unknown-linux-gnu

    runs-on: ${{ matrix.os }}

    steps:
      - uses: actions/checkout@v4

      - name: Setup Bun
        uses: oven-sh/setup-bun@v1

      - name: Setup Rust
        uses: dtolnay/rust-action@stable
        with:
          targets: ${{ matrix.target }}

      - name: Install dependencies
        run: bun install

      - name: Build
        run: bun run tauri build --target ${{ matrix.target }}

      - name: Upload artifacts
        uses: actions/upload-artifact@v4
        with:
          name: ${{ matrix.target }}
          path: src-tauri/target/${{ matrix.target }}/release/bundle/
```

## Signing and Updates

### Generate Keys

```bash
# Generate Ed25519 key pair
bunx tauri signer generate -w ~/.tauri/keys

# Output:
# - ~/.tauri/keys/tpv-el-haido.key (private)
# - ~/.tauri/keys/tpv-el-haido.key.pub (public)
```

### Configure Updater

```json
// tauri.conf.json
{
  "plugins": {
    "updater": {
      "active": true,
      "endpoints": [
        "https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/latest.json"
      ],
      "pubkey": "dW50cnVzdGVkIGNvbW1lbnQ6...",
      "windows": {
        "installMode": "passive"
      }
    }
  }
}
```
