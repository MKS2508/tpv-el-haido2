# Tech Stack

Complete list of technologies and dependencies used in TPV El Haido.

## Frontend

### Framework and UI

| Dependency | Version | Use |
|------------|---------|-----|
| react | 19.2.4 | UI Framework |
| react-dom | 19.2.4 | DOM Rendering |
| @radix-ui/* | latest | Headless accessible components |
| tailwindcss | 4.1.18 | Utility styles |
| framer-motion | 12.29.2 | Animations |
| lucide-react | 0.563.0 | Icons |

### State and Data

| Dependency | Version | Use |
|------------|---------|-----|
| zustand | 5.0.10 | Global state |
| immer | 10.x | Immutability |
| @tanstack/react-virtual | latest | List virtualization |

### Utilities

| Dependency | Version | Use |
|------------|---------|-----|
| @mks2508/no-throw | latest | Result pattern |
| date-fns | latest | Date manipulation |
| uuid | latest | ID generation |
| clsx | latest | Class conditionals |
| tailwind-merge | latest | Tailwind class merging |

## Sidecars

### aeat-bridge

Node.js sidecar for AEAT communication:

| Dependency | Use |
|------------|-----|
| express | HTTP server |
| soap | SOAP client |
| node-forge | Certificate handling |

### thermal-printer-cli

Node.js sidecar for printing:

| Dependency | Use |
|------------|-----|
| escpos | ESC/POS protocol |
| usb | USB communication |
| serialport | Serial communication |

```bash
# Check vulnerabilities

cargo audit
```
