## Installation by Operating System

<Tabs items={['Windows', 'macOS', 'Linux', 'Raspberry Pi']}>
<Tab value="Windows">

### Windows 10/11 (x64)

<Steps>
### Download the installer

Download the `.msi` or `.exe` file from [GitHub Releases](https://github.com/MKS2508/tpv-el-haido2/releases/latest).

### Run the installer

Double-click the downloaded file and follow the installation wizard.

<Callout type="warning">
  Windows may show a SmartScreen warning. Click "More info" and then "Run anyway".
</Callout>

### Launch the application

Search for "TPV El Haido" in the Start menu or double-click the desktop shortcut.

</Steps>

</Tab>
<Tab value="macOS">

### macOS 11+ (Intel and Apple Silicon)

<Steps>
### Download the DMG

Download the `.dmg` file for your architecture:
- **Intel**: `TPV.El.Haido_x64.dmg`
- **Apple Silicon (M1/M2/M3)**: `TPV.El.Haido_aarch64.dmg`

### Install the application

1. Open the downloaded DMG file
2. Drag "TPV El Haido" to the Applications folder

### First launch

<Callout type="warning">
  On first launch, macOS may block the app. Go to **System Preferences → Security & Privacy** and click "Open Anyway".
</Callout>

</Steps>

</Tab>
<Tab value="Linux">

### Linux (Ubuntu, Debian, Fedora)

<Steps>
### Install DEB package (Ubuntu/Debian)

```bash
# Download the package
wget https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/tpv-el-haido_amd64.deb

# Install
sudo dpkg -i tpv-el-haido_amd64.deb

# Resolve dependencies if needed
sudo apt-get install -f
```

### Or use AppImage (any distro)

```bash
# Download
wget https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/tpv-el-haido.AppImage

# Make executable
chmod +x tpv-el-haido.AppImage

# Run
./tpv-el-haido.AppImage
```

</Steps>

</Tab>
<Tab value="Raspberry Pi">

### Raspberry Pi 4/5 (ARM64)

<Steps>
### Prerequisites

- Raspberry Pi 4 or 5
- Raspberry Pi OS (64-bit)
- Minimum 1GB available RAM

### Install the DEB package

```bash
# Download the ARM64 package
wget https://github.com/MKS2508/tpv-el-haido2/releases/latest/download/tpv-el-haido_arm64.deb

# Install
sudo dpkg -i tpv-el-haido_arm64.deb

# Resolve dependencies
sudo apt-get install -f
```

### Run

```bash
tpv-el-haido
```

<Callout type="tip">
  For optimal performance on Raspberry Pi, consider disabling animations in **Settings → Performance**.
</Callout>

</Steps>

</Tab>
</Tabs>
