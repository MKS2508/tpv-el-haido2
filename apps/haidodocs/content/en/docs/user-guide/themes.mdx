# Themes and Customization

TPV El Haido includes a complete theme system to adapt the interface to your business.

## Available Themes

The application includes 6 predefined themes:

| ID | Name | Style | Ideal for |
|----|------|-------|-----------|
| `amethyst-haze` | Amethyst Haze | Elegant violet | Sophisticated venues |
| `bubble` | Bubble | Vibrant pink | Youthful atmosphere |
| `restaurant-professional` | Restaurant Professional | Warm gold | Classic restaurants |
| `modern-cafe` | Modern Cafe | Minimalist green | Modern cafes |
| `night-bar` | Night Bar | Dark with neon | Night bars |
| `accessibility-high-contrast` | High Contrast | Black/White | Maximum readability |

## Change Theme

<Steps>

### Access Settings

1. In the side menu, click **Settings**
2. Go to the **Appearance** or **Theme** section

### Select Theme

1. You'll see a preview of each theme
2. Click the theme you prefer
3. The change applies immediately

### Choose Mode

Each theme supports light and dark mode:
- **Light**: Light backgrounds, dark text
- **Dark**: Dark backgrounds, light text
- **System**: Follows operating system preference

</Steps>

## Light/Dark Mode

### Manual Change

1. In **Settings → Appearance**
2. Select: **Light**, **Dark** or **System**

### Quick Change

- Click the sun/moon icon in the top bar
- Or use the keyboard shortcut (if configured)

### Follow System

When you select "System":
- Daytime: light mode
- Nighttime: dark mode
- Follows your operating system settings

## Touch Optimizations

Themes include optimizations for touch screens:

### Button Sizes

| Element | Size | CSS Variable |
|---------|------|--------------|
| Standard | 44×44px | `--touch-target-size` |
| Large | 48×48px | `--touch-target-large` |
| Extra large | 56×56px | `--touch-target-xl` |

These sizes follow the [Apple Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/).

### Spacing

Themes include extra spacing between elements to prevent accidental touches on touch screens.

## Accessibility

### High Contrast Mode

The **High Contrast** theme is designed for:
- Users with visual difficulties
- High-light environments
- Low-quality screens

### Respects System Preferences

TPV El Haido automatically detects:
- `prefers-reduced-motion`: Disables animations
- `prefers-contrast: high`: Enables additional borders
- `prefers-color-scheme`: Light/dark mode

### Keyboard Navigation

All themes maintain:
- Visible focus for keyboard navigation
- Sufficient contrast per WCAG 2.1
- Readable text sizes

## Tips

### Choosing the Right Theme

| Your Business | Recommended Theme |
|---------------|-------------------|
| Classic restaurant | Restaurant Professional |
| Modern cafe | Modern Cafe |
| Cocktail bar | Night Bar |
| Youthful venue | Bubble |
| Accessibility | High Contrast |
| Elegant/boutique | Amethyst Haze |

### Performance

- On slow devices, consider using the High Contrast theme (fewer effects)
- Dark mode can save battery on OLED screens

### Consistency

- Choose a theme and stick with it so your team gets familiar
- Dark mode is useful for night shifts
