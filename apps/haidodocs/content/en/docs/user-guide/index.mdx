## Main Screen

The TPV El Haido interface is organized into sections accessible from the side menu:

| Section | Description |
|---------|-------------|
| **Home** | Main panel with daily summary |
| **New Order** | Create orders and add products |
| **History** | View previous orders |
| **Products** | Manage catalog |
| **Customers** | Customer database |
| **Invoices** | AEAT invoices and statuses |
| **Settings** | System configuration |
