# Changelog

History of all versions and important changes in TPV El Haido.

  Complete releases are available on [GitHub Releases](https://github.com/MKS2508/tpv-el-haido2/releases).

## Format

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.0.0/):

- **Added**: New features
- **Changed**: Changes to existing features
- **Deprecated**: Features that will be removed soon
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Security updates

## [0.1.0] - 2024-XX-XX

### Added
- PIN-based user login system
- Complete product and category management
- Order/ticket management
- Customer management with tax data
- Order history with filters
- AEAT VERI*FACTU integration
- ESC/POS thermal ticket printing
- Light/dark theme system
- Multiple storage modes (SQLite, HTTP, IndexedDB)
- Multi-platform support (Windows, macOS, Linux, Raspberry Pi)
- Automatic update system

### Technical
- Tauri 2.0 backend with Rust
- React 19 + TypeScript frontend
- Global state with Zustand + Immer
- UI with Radix UI + shadcn/ui + Tailwind CSS
- Embedded SQLite database
- Sidecars for AEAT and thermal printer
